1. To allow downloading files from GDrive, first need to set up an environment , for that refer link below
https://developers.google.com/drive/api/quickstart/python

2. Once app is created with OAuth 2.0 client id,
Save the downloaded JSON file as client_secrets.json, and move the file to your current working directory

3. install required packages by running command
pip3 install -r requirement.txt

4. change local_folder_path and folder_id_to_download in gdrive_download.py file

5. run the gdrive_download.py file using
python3 gdrive_download.py

6. for first time it prompts you to authorize access at google
